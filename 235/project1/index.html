Infinite Shooter

This game was another school project. Within this project I worked alone witin Unity, using C# to program the game logic. It has a very similar style to my bullet-hell game, but I built in many new systems that made my work more efficent. I added pools of enemies and bullets, which made the game much more efficient. I also experimented with random spawning and respawning of enemies. As well as implenting animations for collisions. Live Build

Test of Wills


Test of Wills was a class group project that I worked on in a team of 4. Within this game the player must survive against different bosses in a bullet-hell like environment. The game was programmed in C#, using Monogame as the engine. I personally organized and led the group, designing the architecture of the game. I also did a majority of the programming, including game states, data loading and saving, and overall gameplay logic.
