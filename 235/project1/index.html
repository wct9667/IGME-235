Force of Wills



Force of Wills is a bullet hell style game that was developed during my second semester at RIT as a group project. It was programmed in C# using the Monogame Engine. Players must survive multiple boss fights as quickly as possible to win the game.

My responsibilities were in organizing the structure of the game, as well as being the main programmer behind most of the games systems.

Test of Wills



Force of Wills was a class group project that I worked on in a team of 4. Within this game the player must survive against different bosses in a bullet-hell like environment. The game was programmed in C#, using Monogame as the engine. I personally organized and led the group. While also contributing the main portion of programming of the game, including game states, data loading and saving, and overall gameplay logic.
