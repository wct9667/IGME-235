SHUMP Infinite Shooter(Will Add More)

It has a very similar style to my previus game, but I build in many new systems that made my work more efficent. I added pools of enemies and bullets, meaning objects were dynamically being set active from a pool of objects. Rather than being created and destroyed. This made the game more efficient. Test of Wills was a class group project that I worked on in a team of 4. Within this game the player must survive against different bosses in a bullet-hell like environment. The game was programmed in C#, using Monogame as the engine. I personally organized and led the group, designing the architecture of the game. I also did a majority of the programming, including game states, data loading and saving, and overall gameplay logic.
